## Anakonda

Anakonda is a lightning‑fast Python web framework built for developers who want to create APIs and web applications effortlessly. It combines speed with a clean, intuitive file structure, so you can focus on building instead of boilerplate. Take a look at how simple its syntax can be.

## A minute of context

Anakonda is built directly on the [ASGI](https://asgi.readthedocs.io/) specification, runs on Uvicorn (or Hypercorn/Daphne) in production, and integrates with the existing Python ecosystem like SQLAlchemy, Pydantic, Jinja2 rather than reinventing it.


## Installation

```bash
pip install anakonda.py
```

Anakonda comes in with a built-in CLI (command-line interface) for the framework to ease the use, so to confirm the installation was successful, run:
```bash
konda-admin
```

Now you're all set, it's installed on your machine!
